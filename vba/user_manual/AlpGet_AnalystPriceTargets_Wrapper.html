AlpGet_AnalystPriceTargets Function

The AlpGet_AnalystPriceTargets function retrieves the analyst price targets for a given stock ticker. It uses an underlying wrapper function to send a request to a specified API and writes the received data to an Excel sheet.

Usage

To use the AlpGet_AnalystPriceTargets function, provide the stock ticker and your API api_token. The function will return the price targets data and populate the Excel sheet starting from the current active cell.

Example

Dim result As String result = AlpGet_AnalystPriceTargets("AAPL", "your_api_token_here")

Parameters

  • ticker: The stock ticker symbol you want to get the price targets for (e.g., "AAPL" for Apple Inc.).
  • api_token: Your API token for authentication.

Output

The function will fill the Excel sheet starting from the active cell with the following data:

  • current: The current price target.
  • high: The highest price target.
  • low: The lowest price target.
  • mean: The mean price target.
  • median: The median price target.

Error Handling

If an error occurs during the execution of the function, an error message box will be displayed with the error description.
