AlpGet_CompanyConcept Function

This guide provides information on how to use the AlpGet_CompanyConcept function to retrieve company concept data.

Function Overview

The AlpGet_CompanyConcept function is designed to fetch financial data for a specific company and concept from a local API. The function requires four input parameters:

  • cik: The Central Index Key identifying the company.
  • tag: The financial concept tag you are interested in.
  • taxonomy: The taxonomy identifier.
  • api_token: A security token for API authentication.

The function returns a string composed of the cik, tag and taxonomy concatenated with hyphens.

Example Usage

VBA Example:

Sub RunCompanyConceptExample()
    Dim result As String
    Dim cik As String
    Dim tag As String
    Dim taxonomy As String
    Dim api_token As String
    
    ' Sample input values
    cik = "0000320193"
    tag = "Revenues"
    taxonomy = "us-gaap"
    api_token = "your_api_token_here"
    
    ' Call the AlpGet_CompanyConcept function
    result = AlpGet_CompanyConcept(cik, tag, taxonomy, api_token)
    
    ' Output the result to the immediate window
    Debug.Print "Returned result: " & result
End Sub

This example demonstrates calling the AlpGet_CompanyConcept function to retrieve revenue data for a company with CIK 0000320193.

Response Output

The function will populate the immediate worksheet with the following columns if the data is retrieved successfully:

  • accn: Accession number of the filing.
  • end: End date of the period.
  • filed: Filing date of the document.
  • form: The form type of the SEC filing.
  • fp: Fiscal period.
  • fy: Fiscal year.
  • val: Value for the financial concept.
