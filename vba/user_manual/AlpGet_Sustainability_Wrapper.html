AlpGet_Sustainability Function User Manual

This document provides information on how to use the AlpGet_Sustainability function, which retrieves sustainability data for a given stock ticker.

Function Overview

The AlpGet_Sustainability function serves as a wrapper around the AlpGet_Sustainability_Wrapper function. It connects to a local API to fetch sustainability data for a specified stock ticker, using an API token for authentication.

Function Signature

Function AlpGet_Sustainability(ticker As String, api_token As String) As String

Parameters

  • ticker: String - The stock ticker symbol you want to retrieve sustainability data for.
  • api_token: String - The API token required to authenticate the request.

Return Value

The function returns a String in the format ticker-api_token.

Examples

Example 1: Basic Usage

Dim result As String
result = AlpGet_Sustainability("AAPL", "your_api_token_here")

This example retrieves the sustainability data for the "AAPL" stock ticker using the provided API token. The result is stored in the result variable as a string.

Example 2: Using in a Subroutine

Sub GetSustainabilityData()
    Dim ticker As String
    Dim api_token As String
    Dim output As String

    ticker = "AAPL"
    api_token = "your_api_token_here"
    output = AlpGet_Sustainability(ticker, api_token)

    MsgBox "Retrieved: " & output
End Sub

This example demonstrates how the function can be utilized within a VBA subroutine to get sustainability data for the "AAPL" ticker symbol.

Error Handling

The AlpGet_Sustainability_Wrapper function includes error handling, which displays a message box if an error occurs during the data retrieval process.
