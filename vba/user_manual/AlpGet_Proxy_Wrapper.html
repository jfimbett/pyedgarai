AlpGet_Proxy Function User Manual

The AlpGet_Proxy function provides a way to request and retrieve data corresponding to a specific ticker symbol using a local proxy server. This function uses an API token for authentication and returns the data associated with the ticker if available.

Function Details

Function AlpGet_Proxy(ticker As String, api_token As String) As String

This function acts as a wrapper to retrieve and return the data for a given ticker symbol. Internally, it calls another function AlpGet_Proxy_Wrapper to perform the actual data retrieval and parsing.

Parameters

  • ticker: A string denoting the symbol of the desired stock.
  • api_token: A string representing the user's API token for authentication.

Usage Examples

Example 1

Retrieve data for the ticker "AAPL" using the API token "your_api_token".


Dim data As String
data = AlpGet_Proxy("AAPL", "your_api_token")
' Use the retrieved data (display it within the workbook)

Example 2

Retrieve stock data for "GOOG" with a specific API token.


Dim stockData As String
stockData = AlpGet_Proxy("GOOG", "your_api_token")
' Insert the stock data into the desired worksheet cell

Technical Details

The function constructs a URL for a proxy server endpoint, sends an HTTP GET request, and retrieves a JSON response. The response is parsed to extract the relevant data associated with the requested ticker. Any error during the request or parsing is captured and displayed through a message box.

Error Handling

If an error occurs during the execution of the function, a message box will display the error description, allowing the user to identify and troubleshoot the issue.
