User Manual for AlpGet_RevenueEstimate Function

The AlpGet_RevenueEstimate function in VBA is designed to obtain revenue estimates for a given stock ticker. It does so by sending an HTTP request to an API and processes the JSON response to populate data into an Excel sheet starting from the active cell.

Function Purpose

The main purpose of the AlpGet_RevenueEstimate function is to retrieve revenue estimates for a specific ticker symbol using an API token for authentication. The results are directly inputted into an Excel worksheet.

Parameters

  • ticker (String): The stock ticker symbol for which the revenue estimate is desired.
  • api_token (String): A valid API token to authenticate the request.

Return Value

The function returns a string in the format ticker-api_token, indicating that the request has been processed.

Usage Example


Dim revenueEstimate As String
revenueEstimate = AlpGet_RevenueEstimate("AAPL", "your_api_token_here")

In this example, the function retrieves the revenue estimates for Apple Inc. ("AAPL") using the provided API token and populates the results into the worksheet starting from the current active cell.

Error Handling

If an error occurs during the process, a message box will display the error description along with the line number where the error occurred.
