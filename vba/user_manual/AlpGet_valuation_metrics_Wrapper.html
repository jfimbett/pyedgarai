AlpGet Valuation Metrics Function User Manual

Overview

The AlpGet_valuation_metrics function is designed to retrieve valuation metrics for given tickers using an API token. It fetches a variety of financial metrics and populates them into an Excel worksheet.

Function Signatures

AlpGet_valuation_metrics(tickers As String, api_token As String) As String

Parameters

  • tickers: A comma-separated string of stock tickers to retrieve valuation metrics for.
  • api_token: A string that represents the API token required for authentication.

How It Works

The function makes an HTTP GET request to a local server endpoint to retrieve valuation metrics. The data is then parsed from JSON format, and both average multiples and raw variables for each ticker are populated into the active Excel worksheet.

Output

The function writes multiple financial metrics like Price to Earnings, Market Cap, and more into the active cells and columns of an Excel worksheet. It returns a completion message with the tickers processed.

Examples

Here are some examples of how to use the AlpGet_valuation_metrics function in VBA:


Dim result As String
Dim tickers As String
Dim api_token As String

tickers = "AAPL,MSFT,GOOGL"
api_token = "your_api_token_here"

result = AlpGet_valuation_metrics(tickers, api_token)
MsgBox result
