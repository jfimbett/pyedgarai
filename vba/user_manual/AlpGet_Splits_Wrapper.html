User Manual for AlpGet_Splits Function

Description

The AlpGet_Splits function is used to fetch stock split data for a given ticker symbol using an API token. The data is written directly to the active Excel sheet, starting at the active cell.

Parameters

  • ticker: A string representing the stock ticker symbol.
  • api_token: A string representing the API authentication token.

Return Value

The function returns a string concatenating the ticker and api_token used in the request.

Example Usage

Here's how you can use the AlpGet_Splits function in your VBA code:

Dim result As String
result = AlpGet_Splits("AAPL", "your_api_token_here")

In this example, replace "AAPL" with your desired stock ticker symbol and "your_api_token_here" with your actual API token. The function will populate the active Excel sheet starting at the active cell with the date and stock split data.

Note

Ensure that your Excel environment has JSON parsing capabilities enabled, such as by adding a JSON converter library.
