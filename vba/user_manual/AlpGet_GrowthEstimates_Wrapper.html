AlpGet_GrowthEstimates Function Manual

This manual describes how to use the AlpGet_GrowthEstimates function to retrieve growth estimates for a specified company ticker using an API token.

Function Overview

The AlpGet_GrowthEstimates function is a wrapper that calls another function to send a request to a specified API endpoint, retrieve growth estimates for a given ticker, and populate an Excel sheet with the returned data.

Function Signature

Function AlpGet_GrowthEstimates(ticker As String, api_token As String) As String

Parameters

  • ticker: A String representing the stock ticker symbol for which growth estimates are to be retrieved.
  • api_token: A String containing the API token for authorization.

Return Value

Returns a String indicating the ticker and API token used in the request. It writes the results directly to the current Excel worksheet starting from the active cell.

Usage Example

' Example usage in the Excel VBA environment:
Dim result As String
result = AlpGet_GrowthEstimates("AAPL", "your_api_token_here")
MsgBox "Growth estimates for: " & result

Dependencies

This function requires a local server running at http://localhost:5000 which can respond to GET requests for /growth_estimates endpoint.

Error Handling

If an error occurs during the execution of the function (such as network issues or invalid ticker/API token), a message box will display an error message detailing the problem encountered.
