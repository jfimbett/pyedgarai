AlpGet_Session Function User Manual

Description

The AlpGet_Session function retrieves session data for a specified ticker symbol using a provided API token. It obtains the data by making an HTTP GET request to a local server, parses the JSON response, and writes the data to the Excel sheet starting from the cell immediately below the active cell.

Parameters

  • ticker: A string representing the financial instrument's ticker symbol.
  • api_token: A string representing the API authentication token.

Return Value

Returns a string "Session Retrieved" if successful. If an error occurs, an error message is displayed.

Examples

' Example Usage of AlpGet_Session in VBA/Excel
Sub ExampleUsage()
Dim ticker As String
Dim apiToken As String

ticker = "AAPL"
apiToken = "your_api_token_here"

MsgBox AlpGet_Session(ticker, apiToken)
End Sub
