AlpGet_SecFilings Function User Manual

Overview

The AlpGet_SecFilings function retrieves SEC filings for a specified stock ticker using an API token. It processes and writes the data to an Excel worksheet starting from the active cell.

Function Signature

Function AlpGet_SecFilings(ticker As String, api_token As String) As String

Parameters

  • ticker (String): The stock ticker symbol to fetch SEC filings for.
  • api_token (String): The API token required to authenticate the request.

Return Value

Returns a string composed of the input ticker and api_token separated by a hyphen.

Usage Example

' Declare variables
Dim ticker As String
Dim api_token As String
Dim result As String

' Initialize variables
ticker = "AAPL"
api_token = "your_api_token_here"

' Call the function
result = AlpGet_SecFilings(ticker, api_token)

' Output the result
MsgBox result

Example Usage in Excel VBA

This is how you would typically use the function in an Excel VBA module:

Sub FetchFilings()
    ' Set ticker and API token
    Dim ticker As String
    Dim api_token As String
    ticker = "TSLA"
    api_token = "your_api_token_here"
    
    ' Fetch SEC filings
    Dim output As String
    output = AlpGet_SecFilings(ticker, api_token)
    
    ' Display the result
    MsgBox "Filings fetched for: " & output
End Sub

Note: Ensure that your Excel workbook has access to MSXML2.XMLHTTP and a JSON parser to work with JSON responses.
