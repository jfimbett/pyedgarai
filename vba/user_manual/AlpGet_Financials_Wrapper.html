User Manual for AlpGet_Financials

This document provides a guide for using the AlpGet_Financials function. This function is used to retrieve financial data for a given stock ticker from a specified API endpoint.

Function Description

The AlpGet_Financials function is a high-level wrapper that retrieves financial data for a stock ticker and places it into the current Excel worksheet starting from the active cell. This data is obtained by querying a local API endpoint using the provided ticker symbol and API token.

Parameters

  • ticker: A string representing the stock ticker symbol (e.g., "AAPL" for Apple Inc.).
  • api_token: A string representing the API token used for authentication when accessing the financial data.

Returns

The function returns a string "Success" if the operation was successful. In case of an error, a message box will display the error description.

Usage Examples

Example 1: Retrieve Financial Data for Apple Inc.


Sub GetAppleFinancials()
    Dim result As String
    result = AlpGet_Financials("AAPL", "your_api_token")
    If result = "Success" Then
        MsgBox "Financial data retrieved successfully."
    Else
        MsgBox "Failed to retrieve data."
    End If
End Sub

Example 2: Retrieve Financial Data for Microsoft Corp.


Sub GetMicrosoftFinancials()
    Dim result As String
    result = AlpGet_Financials("MSFT", "your_api_token")
    If result = "Success" Then
        MsgBox "Financial data retrieved successfully."
    Else
        MsgBox "Failed to retrieve data."
    End If
End Sub
Note: Ensure that your local server is running and accessible at the URL "http://localhost:5000/financials" for the function to work correctly.
