User Manual for AlpGet_QuarterlyCashFlow Function

Description

The AlpGet_QuarterlyCashFlow function is used to retrieve and display quarterly cash flow data for a specified stock ticker. It sends a request to a local server API endpoint and populates the Excel sheet starting from the active cell with the retrieved data.

Function Signature

Function AlpGet_QuarterlyCashFlow(ticker As String, api_token As String) As String

Parameters

  • ticker: String - The stock ticker symbol for the company you wish to retrieve cash flow data for.
  • api_token: String - Your API token to authorize the request.

Return Value

The function returns a String representing the concatenation of the ticker and api_token if the operation is successful.

Usage

Example 1
Sub GetCashFlowData()
    Dim response As String
    response = AlpGet_QuarterlyCashFlow("AAPL", "your_api_token_here")
    MsgBox "Result: " & response
End Sub

Error Handling

The function includes basic error handling and will display a message box with the error description if an error occurs.
