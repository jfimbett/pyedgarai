User Manual for AlpGet_QuarterlyFinancials

Description

The AlpGet_QuarterlyFinancials function retrieves quarterly financial data for a given company ticker and API token and displays it in the active Excel sheet.

Function Signature

Function AlpGet_QuarterlyFinancials(ticker As String, api_token As String) As String

Parameters

  • ticker (String): The stock ticker symbol of the company you want to retrieve financial data for.
  • api_token (String): The API token required for authentication with the financial data provider.

Returns

A string formatted as "ticker-api_token" to confirm the function call.

Usage Example

To use the function, enter the following code in a VBA module:

Dim result As String
result = AlpGet_QuarterlyFinancials("AAPL", "YourApiTokenHere")
MsgBox result

Notes

  • The function writes financial data directly to the active Excel sheet starting from the active cell.
  • Ensure the API service is running locally on http://localhost:5000.
  • The function uses a helper method AlpGet_QuarterlyFinancials_Wrapper to handle HTTP requests and JSON parsing.
  • Error handling is included and any errors encountered are displayed using a message box.
