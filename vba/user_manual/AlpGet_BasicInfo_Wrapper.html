User Manual for AlpGet_BasicInfo Function

Introduction

The AlpGet_BasicInfo function is designed to retrieve basic information for a specified stock ticker using an API. The function wraps around another function and facilitates an easy way to get and display stock information within an Excel spreadsheet.

Function Signature

Function AlpGet_BasicInfo(ticker As String, api_token As String) As String

Parameters

  • ticker: String - The stock ticker symbol for the desired company.
  • api_token: String - The API access token required for authenticated requests.

Usage

This function is executed within an Excel VBA environment. It sends a request to an API to retrieve and display stock data. Headings and corresponding data for the ticker will be written into the active worksheet, starting from the active cell.

Example

Sub RetrieveStockInfo()
    Dim ticker As String
    Dim apiToken As String
    Dim result As String

    ticker = "AAPL"
    apiToken = "your_api_token"
    
    result = AlpGet_BasicInfo(ticker, apiToken)
    
    MsgBox "Retrieved Information for: " & result
End Sub

Error Handling

The function includes error handling to display a message box if an error occurs during the data retrieval process. It shows the error description and line number.

Notes

- Ensure that the API server is running and accessible at http://localhost:5000.

- Proper credentials must be passed as api_token to authenticate the API requests.

- If the data key does not exist in the response, the function will exit without writing any data.
