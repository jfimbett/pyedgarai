User Manual for AlpGet_Cik_Names Function

Description

The AlpGet_Cik_Names function is a wrapper for the AlpGet_Cik_Names_Wrapper function. It retrieves a list of CIK (Central Index Key) identifiers along with corresponding names from a local API, and populates this data into an Excel sheet starting from the currently active cell.

Usage

To use this function, you will need to invoke it from an Excel VBA environment. Ensure that the API server is running on localhost:5000 and is capable of providing CIK data in JSON format.

Parameters

  • api_token (String): A token used for API authentication.

Return Value

Returns the api_token that was passed to the function.

Examples


Sub RetrieveCIKNames()
    Dim token As String
    token = "your_api_token_here"
    Call AlpGet_Cik_Names(token)
End Sub

Error Handling

If the function encounters an error (e.g., network issues or JSON parsing errors), a message box will display with the error description and line number.

Dependencies

  • MSXML2.XMLHTTP for HTTP requests.
  • JSON parser, such as JsonConverter, to handle JSON response parsing.
  • Excel environment for execution.

Note

Please ensure your Excel has the necessary permissions and is set up to allow macros and external data connections.
