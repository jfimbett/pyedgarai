AlpGet_Dividends Function User Manual

Function Overview

The AlpGet_Dividends function retrieves dividend information for a specified stock ticker using an API. The function fetches the data and populates it into an Excel worksheet starting from the active cell.

Function Signature

Function AlpGet_Dividends(ticker As String, api_token As String) As String

Parameters

  • ticker: A String representing the stock ticker symbol.
  • api_token: A String representing the API token for authentication.

Return Value

Returns a String concatenation of ticker and api_token as "ticker-api_token".

Usage

To use the AlpGet_Dividends function, call it in your VBA code as shown below:

Sub ExampleUsage()
    Dim result As String
    result = AlpGet_Dividends("AAPL", "your_api_token")
    MsgBox "Data Retrieved for: " & result
End Sub

Ensure that the current active cell in the Excel worksheet is set where you want the data to begin populating. The function will output two headers: Date and Dividends, followed by the corresponding data.

Error Handling

If an error occurs during the API request or JSON parsing, a message box will display the error description and the line number where the error occurred.
