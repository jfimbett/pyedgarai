AlpGet Stocks Data User Manual

Function Overview

The AlpGet_stocks_data function retrieves adjusted closing prices for a list of stock tickers within a specified date range and writes the data to an Excel worksheet starting from the active cell. It requires an API token for authentication.

Function Signature

Function AlpGet_stocks_data(tickers As Variant, start_date As String, end_date As String, api_token As String) As String

Parameters:

  • tickers: An array of stock ticker symbols to fetch data for.
  • start_date: The start date for the data retrieval period, formatted as "YYYY-MM-DD".
  • end_date: The end date for the data retrieval period, formatted as "YYYY-MM-DD".
  • api_token: Your API token for authentication.

Returns:

  • A string representing the tickers and date range used in the format "tickers-start_date-end_date".

Usage Example

Below is an example of how to use the AlpGet_stocks_data function in an Excel VBA context.

' Define the parameters
Dim tickers As Variant
tickers = Array("AAPL", "GOOG", "MSFT")

Dim start_date As String
start_date = "2023-01-01"

Dim end_date As String
end_date = "2023-01-31"

Dim api_token As String
api_token = "your_api_token_here"

' Call the function from a module or VBA
Dim result As String
result = AlpGet_stocks_data(tickers, start_date, end_date, api_token)

' Output result
MsgBox "Data retrieved for: " & result

Error Handling

In case of an error during HTTP request or data processing, the function will display an error message with the description and line number, ensuring users are aware of any issues that arise.
