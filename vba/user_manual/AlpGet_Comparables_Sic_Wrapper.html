AlpGet_Comparables_Sic Function Manual

Description

The AlpGet_Comparables_Sic function retrieves comparable company data for a given CIK (Central Index Key) using an API token. It displays the data in the active Excel worksheet starting from the current active cell. It supports both dictionary and array of dictionaries as JSON responses from the API.

Function Signature

Function AlpGet_Comparables_Sic(cik As String, api_token As String) As String

Parameters

  • cik (String): The Central Index Key for which the comparable data is requested.
  • api_token (String): The token for authenticating API requests.

Returns

A concatenated string of the format "cik=[cik]&api_token=[api_token]". Note that this is primarily used internally to evaluate the wrapper function.

Examples of Usage

Here are examples of how the function can be used in an Excel VBA environment:

Example 1: Using the Function Directly

Sub GetComparablesData()
    Dim cik As String
    Dim apiToken As String
    Dim result As String

    cik = "0001067983" ' Example CIK
    apiToken = "your_api_token_here"

    ' Call the function and store the result
    result = AlpGet_Comparables_Sic(cik, apiToken)

    ' Display the request parameters used
    MsgBox "Request made with parameters: " & result
End Sub

Example 2: Using the Wrapper Function

Sub GetComparablesDataUsingWrapper()
    Dim cik As String
    Dim apiToken As String
    Dim result As String

    cik = "0001067983" ' Example CIK
    apiToken = "your_api_token_here"

    ' Call the wrapper function and display the result
    result = AlpGet_Comparables_Sic_Wrapper(cik, apiToken)
    MsgBox "Request made with parameters: " & result
End Sub

Error Handling

If an error occurs during the execution of the function (e.g., due to an invalid CIK or API token), a message box will appear displaying the error description and the error line.
