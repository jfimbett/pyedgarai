User Manual for AlpGet_Company_Concept Function

Function Overview

The AlpGet_Company_Concept function is designed to retrieve financial data for a specific company and concept using a web API. It takes four parameters: the company's CIK, a tag representing the financial concept, the taxonomy, and an API token required for authentication.

Function Signature

Function AlpGet_Company_Concept(cik As String, tag As String, taxonomy As String, api_token As String) As String

Parameters

  • cik: The Central Index Key (CIK) for the company, used to identify the company in the SEC's EDGAR database.
  • tag: The specific tag identifying the financial concept you want to retrieve.
  • taxonomy: The taxonomy under which the tag is categorized.
  • api_token: A valid API token required for accessing the web service.

Returns

The function returns a string in the format "cik-tag-taxonomy". This string is a composite identifier of the request parameters.

Usage Example

Below is an example of how you can call the AlpGet_Company_Concept function:

Function CallExample()\n Dim cik As String\n Dim tag As String\n Dim taxonomy As String\n Dim api_token As String\n Dim result As String\n \n cik = "0000320193"\n tag = "Revenue"\n taxonomy = "GAAP"\n api_token = "your_api_token_here"\n \n result = AlpGet_Company_Concept(cik, tag, taxonomy, api_token)\n MsgBox "Result: " & result\n End Function

Notes

  • Ensure the API service is running and accessible at the endpoint specified in the function.
  • Make sure the API token is valid and has the necessary permissions.
  • The function makes use of the AlpGet_Company_Concept_Wrapper for its core operations.
