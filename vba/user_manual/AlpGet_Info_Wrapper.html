AlpGet_Info Function User Manual

Introduction

The AlpGet_Info function is a wrapper function designed to retrieve information about a specific stock ticker using an API. The function utilizes an internal function, AlpGet_Info_Wrapper, to perform an HTTP GET request to an external API and then writes the response data into an Excel sheet.

Function Signature

AlpGet_Info(ticker As String, api_token As String) As String

Parameters

  • ticker: A string representing the stock ticker symbol for which information is to be retrieved.
  • api_token: A string representing the API token required for authentication with the API endpoint.

Usage

The function is to be used within an Excel environment where a cell is actively selected. It will query an API for the given stock ticker information and populate the active sheet starting from the currently active cell with the key-value pairs received as a JSON response.

Example

Suppose you want to retrieve information about the stock with the ticker symbol MSFT using the API token your_api_token_here. Place your cursor in the cell where you want the data to be populated and run the following VBA code:

Dim ticker As String
Dim apiToken As String
Dim result As String

ticker = "MSFT"
apiToken = "your_api_token_here"
result = AlpGet_Info(ticker, apiToken)

If result <> "" Then
    MsgBox "Data retrieved for ticker: " & result
Else
    MsgBox "Failed to retrieve data."
End If

Output

The selected cell will be populated with the header "Key" and "Value", and subsequent rows will list each key-value pair from the API response.

Error Handling

In the event of any error (e.g., network issue, invalid response), a message box will display the error description. Adjust your API token and endpoint settings as necessary.
