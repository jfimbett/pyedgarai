AlpGet_Endpoint Function User Manual

Description

The AlpGet_Endpoint function is a wrapper for the AlpGet_Endpoint_Wrapper function. It is used to construct a URL with given parameters, send an HTTP GET request to this URL, and parse the returned JSON response. Results are written to the active Excel worksheet starting at the currently active cell.

Function Signature

Function AlpGet_Endpoint(var1 As String, var2 As String, token As String) As String

Parameters

  • var1 — A string used as the first query parameter in the endpoint URL.
  • var2 — A string used as the second query parameter in the endpoint URL.
  • token — A string used as an authentication token in the endpoint URL.

Return Value

The function returns a concatenated string of var1, var2, and token separated by hyphens.

Usage Examples

Example 1

Sub TestAlpGetEndpoint()
    Dim result As String
    result = AlpGet_Endpoint("value1", "value2", "myToken")
    MsgBox "Result: " & result
End Sub

Example 2

Sub RetrieveData()
    Dim response As String
    response = AlpGet_Endpoint("data1", "data2", "secureToken")
    If response <> "" Then
        MsgBox "Data retrieved and written to the sheet."
    Else
        MsgBox "Failed to retrieve data."
    End If
End Sub

Error Handling

If an error occurs during the process, an error message will be displayed indicating the type and location of the error.
