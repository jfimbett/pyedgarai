User Manual for AlpGet_EarningsHistory Function

Introduction

The AlpGet_EarningsHistory function in Visual Basic for Applications (VBA) is designed to retrieve the earnings history for a given stock ticker from a local web service. The function uses a separate wrapper, AlpGet_EarningsHistory_Wrapper, to send an HTTP request, parse the JSON response, and populate the Excel sheet with the data.

Function Signature

Function AlpGet_EarningsHistory(ticker As String, api_token As String) As String

Parameters

  • ticker: The stock ticker symbol for which the earnings history will be retrieved.
  • api_token: The API token for authenticating the request to the local web service.

Return Value

The function returns a string in the format "[ticker]-[api_token]" if successful; otherwise, no value is returned.

Usage Example


Sub FetchEarningsHistory()
    Dim earningsInfo As String
    Dim ticker As String
    Dim apiToken As String

    ticker = "AAPL"  ' Example stock ticker for Apple Inc.
    apiToken = "your_api_token_here"  ' Replace with your actual API token

    ' Call the function and store the result
    earningsInfo = AlpGet_EarningsHistory(ticker, apiToken)

    ' Output the result
    MsgBox "Earnings history retrieved for: " & earningsInfo
End Sub

Notes

  • Ensure that the local web server is running and accessible.
  • Make sure you have a valid API token to authorize your requests.
  • The earnings data is directly written to the Excel sheet starting from the active cell position.
