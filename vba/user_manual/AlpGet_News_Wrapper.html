User Manual for AlpGet_News Function

Function Purpose

The AlpGet_News function retrieves financial news articles related to a specified ticker symbol. It uses an API to fetch news data which is then written to an active Excel sheet.

Function Signature

Function AlpGet_News(ticker As String, api_token As String) As String

Parameters

  • ticker (String): The stock ticker symbol for which you want to retrieve news.
  • api_token (String): Your API token for authentication.

Instructions for Use

  1. Ensure that the active cell in your Excel sheet is where you want the news data to begin being written.
  2. Enter your API token and desired ticker symbol.
  3. Call the function AlpGet_News with these parameters in your VBA environment.

Output

The function writes a series of articles to the active Excel sheet starting from the active cell, with the following columns:

  • link: URL to the full article.
  • providerPublishTime: Publish time of the article.
  • publisher: Name of the article publisher.
  • title: Title of the news article.
  • type: Type category of the article.
  • uuid: Unique identifier for the article.

Example Usage

' Example 1: Getting news for Apple Inc.
Dim result As String
result = AlpGet_News("AAPL", "YourApiTokenHere")
Cells(1, 1).Select ' Ensure correct cell selection before running this

' Example 2: Getting news for Microsoft
Dim msResult As String
msResult = AlpGet_News("MSFT", "YourApiTokenHere")
Cells(10, 1).Select ' Change to an empty cell range

Error Handling

If an error occurs during the function execution, a message box will display the error description.
