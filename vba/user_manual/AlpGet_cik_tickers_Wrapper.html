User Manual for AlpGet_cik_tickers Function

The AlpGet_cik_tickers function retrieves financial data related to CIK (Central Index Key) and ticker symbols from a local server and populates the data into an Active Excel Sheet. This function serves as a wrapper around the AlpGet_cik_tickers_Wrapper function, which performs the detailed tasks of data retrieval and sheet population.

Function Signature

Function AlpGet_cik_tickers(api_token As String) As String

Parameters

  • api_token (String): A string representing the API token required to authenticate the request to the local server.

Return Value

The function returns a string indicating the success of the operation. On successful completion, it returns "Request successful". If an error occurs, it will trigger an error message displaying the error information.

Functionality

The AlpGet_cik_tickers_Wrapper function sends a GET request to a local server endpoint to retrieve JSON data containing CIK and ticker symbol pairs. The data is then parsed and populated into the active Excel sheet starting from the current active cell. Headers "CIK" and "Ticker" are added, followed by the data rows.

Usage Example

' Usage of AlpGet_cik_tickers function in VBA
Sub ExampleUse()
    Dim result As String
    Dim apiToken As String
    apiToken = "your_api_token_here"
    result = AlpGet_cik_tickers(apiToken)
    MsgBox result
End Sub

Important Note

Ensure that you have the JSON parser library (JsonConverter) available in your VBA environment. You also need to replace "your_api_token_here" with a valid API token for the request to be successful.
