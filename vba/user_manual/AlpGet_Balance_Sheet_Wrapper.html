AlpGet_Balance_Sheet Function User Manual

Function Purpose

The AlpGet_Balance_Sheet function retrieves the balance sheet data for a specified company ticker using a provided API token. The data is fetched from an API endpoint and populated into the active Excel sheet starting from the current active cell.

Function Signature


Function AlpGet_Balance_Sheet(ticker As String, api_token As String) As String

Parameters

  • ticker: String - The stock ticker symbol for the company.
  • api_token: String - The API token required for authentication.

Return Value

Returns a String that concatenates the ticker and the API token used in the request.

Usage Example


Sub FetchBalanceSheet()
    Dim result As String
    Dim ticker As String
    Dim apiToken As String
    
    ticker = "AAPL"
    apiToken = "your_api_token_here"
    
    ' Call the function and retrieve the balance sheet data
    result = AlpGet_Balance_Sheet(ticker, apiToken)
    
    If result <> "" Then
        MsgBox "Data retrieval successful: " & result
    Else
        MsgBox "Data retrieval failed."
    End If
End Sub

Error Handling

The function includes basic error handling that will display a message box with the error description if an error occurs during execution.
