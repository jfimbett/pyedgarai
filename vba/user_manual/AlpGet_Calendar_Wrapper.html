AlpGet_Calendar Function User Manual

Introduction

The AlpGet_Calendar function accesses financial calendar data for a given stock ticker by sending an HTTP GET request to a local API endpoint, parses the JSON response and writes the data into the Excel sheet starting from the active cell.

Function Signature

Function AlpGet_Calendar(ticker As String, api_token As String) As String

This function takes two parameters:

  • ticker: The stock ticker symbol for which you want to retrieve financial calendar data.
  • api_token: An API token to authorize the request.

Example Usage

This example assumes that the API server is running locally and is accessible.

Sub ExampleUsage()
    Dim ticker As String
    Dim apiToken As String
    Dim response As String

    ticker = "AAPL" ' Apple's stock symbol
    apiToken = "your_api_token_here"

    ' Call the function
    response = AlpGet_Calendar(ticker, apiToken)

    ' Output the response
    MsgBox "Response: " & response
End Sub

Error Handling

If an error occurs during the execution of the function (e.g., if the API is unreachable or the response does not contain expected data), an error message box will be displayed showing the error description and location.

Make sure your API server is running, and that your request parameters (like ticker and API token) are correct to avoid errors.

Dependencies

  • The function uses MSXML2.XMLHTTP for making HTTP requests.
  • JSON response parsing requires a JsonConverter library, which should be included in your VBA project.
