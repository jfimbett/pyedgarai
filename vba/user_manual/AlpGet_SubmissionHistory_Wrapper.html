User Manual for AlpGet_SubmissionHistory Function

Function Overview

The function AlpGet_SubmissionHistory is used to retrieve submission history data based on the CIK (Central Index Key) and an API token. The data is fetched from a specified API endpoint, parsed, and then populated into the active Excel sheet at the current selection point.

Function Syntax

AlpGet_SubmissionHistory(cik As String, api_token As String) As String

Parameters

  • cik: A string representing the Central Index Key of the entity you want to retrieve data for.
  • api_token: A string representing the API token for authentication with the API endpoint.

Return Value

The function returns a string in the format cik-api_token which can be used for verification of the function call.

Examples of Usage

Example 1: Basic Usage


Dim result As String
result = AlpGet_SubmissionHistory("0000320193", "your_api_token_here")
If result <> "" Then
    MsgBox "Submission history retrieved successfully!"
End If

Example 2: Using in an Excel Macro


Sub FetchSubmissionHistory()
    Dim cik As String
    Dim api_token As String
    Dim output As String
    
    cik = "0000320193"
    api_token = "your_api_token_here"
    
    output = AlpGet_SubmissionHistory(cik, api_token)
    
    If output <> "" Then
        MsgBox "Submission history for " & cik & " retrieved successfully!"
    Else
        MsgBox "Failed to retrieve submission history."
    End If
End Sub

Error Handling

Please ensure that both the CIK and API Token are correct. If the function encounters an error, a message box will display the error description and location.
