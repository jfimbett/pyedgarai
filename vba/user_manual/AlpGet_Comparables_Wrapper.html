AlpGet_Comparables Function User Manual

The AlpGet_Comparables function is a VBA function that retrieves financial comparables for a specified company using various criteria. It sends a request to a local web server and returns the results to an Excel worksheet.

Function Parameters

  • cik (String): The CIK code of the company.
  • method (String): The method to be used for comparison.
  • api_token (String): The API token for authentication.
  • industry_digits (String): Industry classification digits.
  • size_interval (String): Size interval for filtering companies.
  • profitability_interval (String): Profitability interval for filtering companies.
  • growth_rate_interval (String): Growth rate interval for filtering companies.
  • capital_structure_interval (String): Capital structure interval for filtering companies.

Returns

A string composed of the cik, method, and api_token values concatenated with hyphens.

Example Usage

Below is an example of how to call the AlpGet_Comparables function in VBA:


Sub ExampleUsage()
    Dim result As String
    result = AlpGet_Comparables("123456", "method1", "token123", "1234", "10-50", "5-15", "10-20", "1-5")
    MsgBox result
End Sub

Explanation

In this example, the function is called with specific parameters. The financial comparables data is retrieved for the company with the specified CIK, using the defined method and filtered by the given financial criteria. The data is then written directly into the active Excel worksheet starting from the active cell.
