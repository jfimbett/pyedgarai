AlpGet Earnings Function User Manual

Function Overview

The AlpGet_Earnings_Wrapper and AlpGet_Earnings are VBA functions designed to fetch earnings data from a local server. The AlpGet_Earnings function is a simplified interface for the AlpGet_Earnings_Wrapper function, which handles the actual data retrieval process.

Function Signatures

  • Function AlpGet_Earnings_Wrapper(ticker As String, api_token As String) As String
  • Function AlpGet_Earnings(ticker As String, api_token As String) As String

Parameters

ticker
A string representing the stock ticker symbol whose earnings data you want to retrieve.
api_token
A string containing the API token required for authentication when accessing the data.

Functionality

The function sends an HTTP GET request to a local server to retrieve earnings data for a specific stock ticker using the provided API token. On successful retrieval, it updates the active Excel cell with the earnings data.

Usage Examples

Example 1: Retrieve Earnings for a Single Ticker

Sub TestEarnings()
    Dim result As String
    result = AlpGet_Earnings("AAPL", "your_api_token")
    MsgBox "Earnings Data Retrieved: " & result
End Sub

Example 2: Writing Earnings Data to Active Cell

Sub WriteEarningsToCell()
    AlpGet_Earnings "GOOGL", "your_api_token"
    ' The data will be written to the active cell in Excel.
End Sub

Error Handling

If an error occurs during the data retrieval process, a message box will display the error description and line number within the AlpGet_Earnings_Wrapper function.

Please ensure that the JSON library for VBA is installed and that your local server is running at http://localhost:5000.
