AlpGet_Ticker Function User Manual

This document provides a user manual for the AlpGet_Ticker function, explaining its purpose, usage, and examples.

Function Overview

The AlpGet_Ticker function is designed to retrieve stock ticker information from a local API endpoint. This function utilizes a wrapper to handle HTTP requests and JSON parsing, ultimately displaying the data in the current active Excel cell.

Function Signature

Function AlpGet_Ticker(ticker As String, api_token As String) As String

Parameters

  • ticker: A string representing the stock ticker symbol to fetch information for.
  • api_token: A string representing the API token required for authentication with the API endpoint.

Return Value

The function returns a string that is the original ticker parameter passed. If data retrieval is successful, the corresponding data is displayed in the active Excel cell.

Example Usage

To use the AlpGet_Ticker function, ensure that your VBA environment is configured to access the necessary external APIs, and the JSON converter library is available. Below is an example of how to use this function in a VBA environment:

' Example usage of AlpGet_Ticker
Sub ExampleUsage()
    ' Declare ticker and API token variables
    Dim sampleTicker As String
    Dim sampleApiToken As String

    ' Assign values to the variables
    sampleTicker = "AAPL"
    sampleApiToken = "your_api_token_here"

    ' Invoke the AlpGet_Ticker function
    Dim result As String
    result = AlpGet_Ticker(sampleTicker, sampleApiToken)

    ' Output result
    MsgBox "Ticker Symbol: " & result
End Sub

Handling Errors

The function AlpGet_Ticker_Wrapper includes error handling to display a message box in case an error occurs during execution, indicating the error description.

Dependencies

This function requires access to a local API running on http://localhost:5000/ and assumes the existence of a JSON converter to parse API responses in VBA.
