User Manual for AlpGet_CleanName Functions

The AlpGet_CleanName function is used to clean a given name through an API call. The function returns a formatted string combining the original name and the API token.

Function Description

Function AlpGet_CleanName(name As String, api_token As String) As String

Parameters:

  • name (String): The name that needs to be cleaned.
  • api_token (String): The API token used for authentication in the API request.

Returns:

  • (String): A string formatted as name-api_token.

Example Usage

VBA Code Example

Sub ExampleUsage()
    Dim rawName As String
    Dim token As String
    Dim cleanedName As String
    
    rawName = "John Doe"
    token = "12345xyz"
    cleanedName = AlpGet_CleanName(rawName, token)
    
    MsgBox "Formatted Result: " & cleanedName
End Sub

This example illustrates how to use the AlpGet_CleanName function within a VBA macro to clean the name "John Doe", using the API token "12345xyz" and display the formatted result using a message box.

Excel Worksheet Example

Suppose you want to use the function in an Excel worksheet. You can create a button that triggers the ExampleUsage macro to utilize the AlpGet_CleanName function.

Error Handling

If an error occurs during the execution of the function, a message box will display the error description.
