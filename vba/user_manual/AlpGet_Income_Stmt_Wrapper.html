AlpGet_Income_Stmt Function

Retrieve and display income statements for a specified company ticker

Introduction

The function AlpGet_Income_Stmt is designed to fetch the income statement for a given company ticker using a specified API token. It processes the data and populates it into an active Excel worksheet.

Function Definitions

AlpGet_Income_Stmt_Wrapper

This function handles the HTTP request to fetch the income statement and processes the JSON response.

AlpGet_Income_Stmt

This function acts as a wrapper that utilizes Evaluate to call AlpGet_Income_Stmt_Wrapper.

Parameters

  • ticker (String): The stock ticker symbol of the company.
  • api_token (String): An API token for authentication with the data source.

Return Value

The function returns a string combining the ticker and the API token.

Error Handling

In case of an error, a message box displays the error description along with the line where the error occurred.

Examples

Example 1: Retrieve Income Statement

Dim response As String
response = AlpGet_Income_Stmt("AAPL", "your_api_token_here")
' After execution, the active sheet will be filled with Apple's income statement data.
' The variable 'response' will hold "AAPL-your_api_token_here".

Example 2: Handling Errors

Dim response As String
On Error Resume Next
response = AlpGet_Income_Stmt("INVALID", "wrong_token")
If Err.Number <> 0 Then
    MsgBox "An error occurred while retrieving the data."
End If
On Error GoTo 0

Notes

Ensure the API endpoint specified in the function is accessible and that the server is running to handle requests.
Activate the cell in the Excel sheet where you want data to be populated before invoking the function.
