AlpGet_AllAccounts Function User Manual

Function Overview

The AlpGet_AllAccounts function is designed to retrieve account data from an API endpoint using a given API token. The data is fetched and arranged in a structured format starting at the currently active cell in an Excel worksheet.

Function Definition

Function AlpGet_AllAccounts(api_token As String) As String
    Dim result As String
    result = Evaluate("AlpGet_AllAccounts_Wrapper(""" & api_token & """)")
    AlpGet_AllAccounts = result
End Function

Parameters

  • api_token: A String representing the API token used for authentication.

Return Value

The function returns a String with the value "All Accounts Data Retrieved" if the operation is successful.

Usage Example

Sub RetrieveAccountsData()
    Dim api_token As String
    Dim result As String
    api_token = "your_api_token_here"
    
    ' Call the function to retrieve account data
    result = AlpGet_AllAccounts(api_token)
    
    ' Display the result
    MsgBox result
End Sub

Error Handling

If any error occurs during the execution of the function, a message box will appear displaying the error description and line number.
