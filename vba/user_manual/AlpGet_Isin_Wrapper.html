User Manual for AlpGet_Isin Function

Description

The AlpGet_Isin function is designed to retrieve the International Securities Identification Number (ISIN) for a given stock ticker. It operates by making a REST API call to a local server endpoint using the provided ticker and API token. The result is displayed in the active Excel cell.

Input Parameters

  • ticker (String): The stock ticker symbol for which the ISIN is requested.
  • api_token (String): A valid API token to authenticate the request.

Output

The function returns a concatenated string comprising the ticker and API token. In case of an error during the API call, a message box will alert the user.

Usage Example


Dim isin As String
isin = AlpGet_Isin("AAPL", "123456")

In this example, the AlpGet_Isin function fetches the ISIN associated with the ticker "AAPL", using the API token "123456". The result is displayed in the currently active Excel cell.

Error Handling

The function includes error handling to alert the user if an error occurs during execution, such as issues with the API server or an invalid response.
