User Manual for AlpGet_QuarterlyBalanceSheet

Function Overview

The AlpGet_QuarterlyBalanceSheet function allows you to retrieve the quarterly balance sheet data for a specified ticker using an API endpoint. The data is then displayed in the active Excel worksheet starting from the currently selected cell.

Function Signature

Function Name: AlpGet_QuarterlyBalanceSheet

Parameters:

  • ticker (String) - The stock ticker symbol for the company.
  • api_token (String) - The API token for authentication.

Returns: (String) - A combination of the ticker and api_token.

Usage

Below are examples of how to use the AlpGet_QuarterlyBalanceSheet function in VBA/Excel:

Example 1:

Dim ticker As String
Dim api_token As String
Dim result As String

ticker = "AAPL"
api_token = "your_api_token_here"

result = AlpGet_QuarterlyBalanceSheet(ticker, api_token)

' The result will contain the ticker and api_token
MsgBox result

Example 2:

Sub RetrieveBalanceSheet()
    Dim ticker As String
    Dim api_token As String
    Dim result As String

    ticker = "GOOG"
    api_token = "your_api_token_here"

    result = AlpGet_QuarterlyBalanceSheet(ticker, api_token)

    ' Display the result in a message box
    MsgBox "Data retrieved for: " & result
End Sub
