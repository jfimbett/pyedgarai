AlpGet_Account Function User Manual

Introduction

The AlpGet_Account function is a VBA function designed to fetch account information from a local server and populate it into an Excel sheet. This guide provides details about the function and examples of its usage.

Function Signature

Function AlpGet_Account(units As String, account As String, frame As String, taxonomy As String, api_token As String) As String

Parameters

  • units: String - Represents the unit of the account information requested.
  • account: String - Denotes the specific account to fetch data for.
  • frame: String - The frame of reference (or time frame) for the account information.
  • taxonomy: String - Categorizes the type of account information required.
  • api_token: String - A token for API authorization.

Return Value

The function returns a string in the format units-account-frame-taxonomy that represents the combined parameters.

Side Effects

The function populates the Excel sheet, starting from the active cell, with account information fetched from a local API, laid out in columns: accn, cik, end, entityName, loc, start, val.

Error Handling

If an error occurs, a message box will display an error description and line number.

Example Usage

Dim result As String
result = AlpGet_Account("USD", "Cash", "2023-Q1", "IFRS", "your_api_token")

The above example calls the AlpGet_Account function with parameters for fetching quarterly cash account information in USD format, using an IFRS taxonomy.

Dependencies

The function requires the MSXML2 library for HTTP requests and a JSON converter to parse responses. It communicates with a local server that's expected to be running at http://127.0.0.1:5000.
