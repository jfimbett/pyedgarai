AlpGet_Quarterly_IncomeStmt Function

Description: This function retrieves the quarterly income statement for a given stock ticker using an API. It uses a JSON parser to process the data and output it to the Excel sheet starting from the active cell.

Function Signature

Function AlpGet_Quarterly_IncomeStmt(ticker As String, api_token As String) As String

Parameters

  • ticker: A string representing the stock ticker symbol.
  • api_token: A string containing the API token required for authentication.

Returns

Returns a string formed by concatenating the ticker and the API token, separated by a hyphen.

Example Usage

Below is an example of how to use the AlpGet_Quarterly_IncomeStmt function in VBA:

Sub ExampleUsage()
    Dim result As String
    Dim ticker As String
    Dim api_token As String

    ticker = "AAPL"
    api_token = "your_api_token_here"

    result = AlpGet_Quarterly_IncomeStmt(ticker, api_token)
    
    MsgBox "Result: " & result
End Sub

Error Handling

The function uses error handling to display any errors that might occur during the execution process using a message box.

Notes

  • Ensure that the API server (localhost:5000) is running and accessible.
  • The API token must be valid for the API call to succeed.
  • The JSON structure returned by the API should have a "data" key containing the financial data.
  • The function requires a JSON parser in VBA, such as VBA-JSON.
