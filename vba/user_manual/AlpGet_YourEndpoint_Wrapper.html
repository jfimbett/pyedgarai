AlpGet_YourEndpoint Function User Manual

Introduction

The AlpGet_YourEndpoint function is a VBA function designed to make an HTTP GET request to a specified endpoint with given parameters. The response is expected to be in JSON format and contain data that will be written to the active Excel sheet.

Function Definition

Function AlpGet_YourEndpoint(param1 As String, param2 As String, param3 As String, api_token As String) As String

Parameters

  • param1 (String): The first parameter to be sent in the request URL.
  • param2 (String): The second parameter to be sent in the request URL.
  • param3 (String): The third parameter to be sent in the request URL.
  • api_token (String): The API token for authentication purposes.

Return Value

This function returns a concatenated string of the parameters in the format param1-param2-param3.

Behavior

The function performs the following operations:

  1. Constructs and sends an HTTP GET request using the provided parameters and API token to a predefined endpoint.
  2. Parses the JSON response.
  3. Writes specific keys from the JSON response data to the Excel sheet, starting from the cell below the current active cell.
  4. Returns a concatenated string of the input parameters.

Usage Example

Sub ExampleUsage()
Dim result As String
result = AlpGet_YourEndpoint("value1", "value2", "value3", "your_api_token")
MsgBox "Result: " & result
End Sub
