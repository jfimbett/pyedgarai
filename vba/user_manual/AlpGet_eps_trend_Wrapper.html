User Manual for AlpGet_eps_trend Function

Overview

The AlpGet_eps_trend function allows users to retrieve EPS trend data for a specific ticker from a local API endpoint. It acts as a wrapper that simplifies the process of sending HTTP requests and placing the response data into an Excel worksheet.

Function Syntax

Function AlpGet_eps_trend(ticker As String, api_token As String) As String
    ' Retrieve the EPS trend data
End Function

Parameters

  • ticker: A string representing the stock ticker symbol for which EPS trend data is requested.
  • api_token: A string that serves as the authentication token required to access the API.

Return Value

The function returns a string in the format "{ticker}-eps_trend" indicating the ticker and the task completed.

Usage Example

Sub GetEPSTrend()
    Dim result As String
    result = AlpGet_eps_trend("AAPL", "your_api_token_here")
    MsgBox result
End Sub

Example Explanation

In the above example, the GetEPSTrend subroutine calls the AlpGet_eps_trend function with the ticker symbol "AAPL" and a specified API token. After execution, a message box displays the result, indicating the EPS trend data for Apple Inc. has been retrieved.

Error Handling

If an error occurs during the process of retrieving data, a message box will display the error description along with the line number where the error happened, allowing users to diagnose and correct issues efficiently.
