User Manual for AlpGet_Capital_Gains

Function Overview

The AlpGet_Capital_Gains VBA function is used to retrieve capital gains data for a specified stock ticker using an API. This function acts as a wrapper around AlpGet_Capital_Gains_Wrapper to provide the desired functionality. The function sends an HTTP request to a local service for retrieving the data and formats the response into the active Excel worksheet.

Function Signature

Function AlpGet_Capital_Gains(ticker As String, api_token As String) As String

Parameters

  • ticker: A String representing the stock ticker symbol for which you want to retrieve capital gains data.
  • api_token: A String representing the API token required for authentication to access the capital gains data.

Return Value

The function returns a String by concatenating the ticker and the API token, primarily for confirmation purposes.

Example Usage

Example 1: Retrieving Capital Gains Data

The example below demonstrates the typical usage of the function within an Excel VBA module:

Sub GetCapitalGainsExample()
    Dim ticker As String
    Dim api_token As String
    Dim result As String
    
    ticker = "AAPL"
    api_token = "your_api_token_here"
    
    result = AlpGet_Capital_Gains(ticker, api_token)
    MsgBox "Data Retrieved: " & result
End Sub

Error Handling

If an error occurs during execution, a message box will display the error description to the user.
