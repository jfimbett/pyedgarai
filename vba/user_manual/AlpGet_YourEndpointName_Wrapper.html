API Function User Manual

Function Overview

The AlpGet_YourEndpointName function allows you to interact with a specified API endpoint by sending a HTTP GET request with parameters and retrieves the JSON response to output the data into the active Excel sheet.

Parameters

  • param1: A string parameter to be appended to the API URL.
  • param2: A string parameter to be appended to the API URL.
  • param3: A string parameter to be appended to the API URL.
  • api_token: The API token string required for authentication.

Return Value

Returns a concatenated string of parameters param1, param2, and param3 separated by hyphens.

Usage Example

Below is an example of how to use the AlpGet_YourEndpointName function in VBA:

Sub TestAPI()
    Dim result As String
    result = AlpGet_YourEndpointName("value1", "value2", "value3", "your_api_token")
    MsgBox result
End Sub

Response Handling

If the API request is successful and the JSON response contains a data array, the function will write each element's key1, key2, and key3 values into the active Excel sheet starting from the cell below the current active cell.

Error Handling

If an error occurs within the function, a message box will display the description of the error for debugging purposes.
