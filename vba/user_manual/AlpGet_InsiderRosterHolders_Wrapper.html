User Manual for AlpGet_InsiderRosterHolders Function

Overview

The AlpGet_InsiderRosterHolders function retrieves a company's insider roster holders by sending a request to a specified API. The data is then parsed and populated into an Excel worksheet starting at the active cell. This function allows you to access information such as names, positions, URLs, transaction details, and ownership data of company insiders.

Syntax

Function AlpGet_InsiderRosterHolders(ticker As String, api_token As String) As String

Parameters

  • ticker: The stock ticker symbol of the company you want to query.
  • api_token: Your API access token for authentication.

Return Value

The function returns a string composed of the ticker and API token used for the request, but its main output is writing the data to the Excel worksheet.

Usage Examples

Example 1

Sub DemoUsage()
    Dim ticker As String
    Dim api_token As String
    Dim result As String
    
    ticker = "AAPL"
    api_token = "your_api_key_here"
    
    result = AlpGet_InsiderRosterHolders(ticker, api_token)
    MsgBox "Data for " & result & " has been retrieved and written to the worksheet."
End Sub

Example 2

Sub AnotherExample()
    Dim result As String
    result = AlpGet_InsiderRosterHolders("GOOGL", "another_api_key")
    If result <> "" Then
        MsgBox "Insider roster holders for Google have been successfully populated."
    Else
        MsgBox "Failed to retrieve data."
    End If
End Sub
