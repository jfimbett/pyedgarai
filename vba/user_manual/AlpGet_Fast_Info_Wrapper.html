AlpGet_Fast_Info VBA Function User Manual

Introduction

The AlpGet_Fast_Info function is used to fetch financial information for a specified stock ticker from an API, using a provided API key. The data is then displayed in Excel, starting from the current active cell.

Function Definition

Function AlpGet_Fast_Info(ticker As String, api_token As String) As String

Parameters

  • ticker (String): The stock ticker for which information is to be fetched.
  • api_token (String): The API token required to authenticate the request.

Return Value

The function returns a string in the format of ticker-api_token, indicating successful completion.

Usage Example

To use the function, simply call it within your VBA code like this:


Sub ExampleUsage()
    Dim result As String
    result = AlpGet_Fast_Info("AAPL", "your_api_token_here")
    MsgBox "Data fetched for: " & result
End Sub

Notes

  • Ensure the API server is running at http://localhost:5000/ with an endpoint /fast_info.
  • The function writes the headers and data to the Excel sheet starting from the active cell.
  • If an error occurs, an error message box will appear displaying the error description and the line number.

Error Handling

The function includes error handling to catch any runtime errors and will display a message box with the error description.

Conclusion

This function facilitates easy retrieval of stock data within an Excel environment, providing a structured way to obtain and insert information directly into your workbook.
