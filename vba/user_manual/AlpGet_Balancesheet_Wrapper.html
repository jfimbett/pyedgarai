AlpGet_Balancesheet Function User Manual

Introduction

The AlpGet_Balancesheet function is designed to retrieve and display the balance sheet data for a specified company ticker from an API endpoint. The main role of this function is to automate the process of fetching financial data and organizing it neatly into your Excel Worksheet.

Function Signature

Function AlpGet_Balancesheet(ticker As String, api_token As String) As String

Parameters

  • ticker: A String representing the stock ticker of the company you want to retrieve balance sheet data for.
  • api_token: A String representing the API token required to authenticate requests to the endpoint.

Return Value

Returns a String that concatenates the ticker and api_token, mainly for testing and identification purposes.

Function Workflow

The function follows these steps:

  1. Constructs a URL using the provided ticker and API token.
  2. Sends a GET request to the specified endpoint.
  3. Parses the JSON response and retrieves the balance sheet data.
  4. Writes the data, starting from the currently active cell in Excel, organized by date and other financial entries in columns.

Usage Example

Here is an example on how to use the AlpGet_Balancesheet function in VBA:

' Example usage of the AlpGet_Balancesheet function
Sub FetchCompanyBalanceSheet()
    Dim result As String
    Dim ticker As String
    Dim apiToken As String
    
    ' Set ticker and API Token
    ticker = "AAPL"
    apiToken = "your_api_token_here"
    
    ' Call the function with the ticker and API Token
    result = AlpGet_Balancesheet(ticker, apiToken)
    
    ' Output the result for confirmation
    MsgBox "Balance sheet data has been fetched for: " & result
End Sub

Error Handling

If an error occurs during the execution of the function, such as an issue with the API request or parsing the JSON response, a message box will display an error message with a description.

Ensure that the provided API endpoint is reachable and the API token is valid to avoid errors during the operation.
