User Manual for AlpGet_Options

Function Overview

The AlpGet_Options function is designed to interact with a local web server to fetch option expiration dates for a specified stock ticker. It retrieves this data by calling a REST API endpoint and writes the expiration dates into the currently active Excel sheet, starting from the active cell position. This function wraps around another internal function AlpGet_Options_Wrapper, which performs the HTTP request and processes the response.

Usage

The AlpGet_Options function takes two parameters:

  • ticker: The stock ticker symbol for which you want to retrieve option expiration dates.
  • api_token: An API token for authentication.

Example

Here is an example of how to use the AlpGet_Options function in VBA:

Sub ExampleUsage()
    Dim result As String
    result = AlpGet_Options("AAPL", "your_api_token_here")
    MsgBox "Operation completed for: " & result
End Sub

Response Handling

Upon successful execution, the expiration dates will be written to an Excel sheet starting from the active cell. This function reports the progress by returning the concatenated ticker and api_token string upon completion.

Error Handling

If an error occurs during the execution, a message box displaying the error description will appear. This might happen if there is an issue with the HTTP request or if the data could not be retrieved from the server.

Please, ensure that the local server is running and accessible on http://localhost:5000.
