AlpGet_cik_sic Function User Manual

This manual explains the usage of the AlpGet_cik_sic function which retrieves data from an API and writes specific information to an Excel worksheet.

Function Purpose

The AlpGet_cik_sic function is designed to interact with an API endpoint that returns a JSON object containing CIK (Central Index Key) and SIC (Standard Industrial Classification) data. The function processes this data and outputs it to the currently active Excel sheet, writing each key-value pair into separate cells.

Parameters

  • api_token (String): The authentication token required to access the API.

Return Value

The function returns a String indicating the status of the operation. It will return "Completed" upon successful execution or an error message if something goes wrong.

Usage Example

The following VBA example demonstrates how to use the AlpGet_cik_sic function:

Sub ExampleUsage()
  Dim status As String
  Dim apiToken As String

  ' Set your API token here
  apiToken = "your_api_token_here"

  ' Call the function and store the result
  status = AlpGet_cik_sic(apiToken)

  ' Output the result to the user
  MsgBox "Function execution status: " & status
End Sub

Notes

  • Ensure that the API endpoint http://localhost:5000/cik_sic is accessible and that you have a valid API token.
  • Always run the function with the active cell set where data output should begin.
  • In case of an error, a message box will display the error description and line number.
