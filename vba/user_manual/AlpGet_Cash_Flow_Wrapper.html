AlpGet_Cash_Flow Function User Manual

Function Purpose

The AlpGet_Cash_Flow function is a wrapper that retrieves and displays cash flow data for a specified ticker symbol via an API. The data is presented directly in an Excel worksheet starting from the currently active cell.

Function Signature

Function AlpGet_Cash_Flow(ticker As String, api_token As String) As String

Parameters:

  • ticker: A string representing the ticker symbol of the stock.
  • api_token: An authorization token required to access the API.

Usage Instructions

  1. Ensure your Excel worksheet is open and the desired starting cell is selected.
  2. Call the function with appropriate ticker and api_token arguments.
  3. The Excel worksheet will be populated with cash flow data starting from the currently active cell.

Examples


Sub ExampleUsage()
    Dim result As String
    result = AlpGet_Cash_Flow("AAPL", "your_api_token_here")
    MsgBox "Cash Flow Data Retrieved for: " & result
End Sub

This example gets the cash flow data for the ticker "AAPL" and displays a message box indicating the retrieval. Make sure you replace your_api_token_here with your actual API token.

Error Handling

If an error occurs during the data retrieval process, a message box will display the error description to help debug the issue.
