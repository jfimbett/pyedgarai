AlpGet Recommendations User Manual

Overview

The AlpGet_Recommendations_Wrapper and AlpGet_Recommendations functions allow users to fetch stock recommendation data from a specified endpoint. This functionality is useful for retrieving investment recommendations based on the provided ticker symbol.

    ' Parameters:
    '   ticker - The stock ticker symbol

    ' Returns:
    '   A string formatted as "ticker-api_token"

Usage Examples

Example 1: Simple Usage

Below is an example of how to use the AlpGet_Recommendations function in VBA:

Sub FetchRecommendationsExample()
    Dim result As String
    result = AlpGet_Recommendations("AAPL", "your_api_token")
    MsgBox "Result: " & result
End Sub

Example 2: Display Results in Excel

Retrieve recommendations and display them in an Excel worksheet:

Sub DisplayRecommendations()
    Dim ticker As String, apiToken As String
    ticker = "MSFT"
    apiToken = "your_api_token"
    
    ' Call the function and display results starting from the active cell
    AlpGet_Recommendations_Wrapper ticker, apiToken
End Sub
