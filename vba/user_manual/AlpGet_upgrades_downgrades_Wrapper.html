AlpGet_upgrades_downgrades: User Manual

This manual provides information on how to use the AlpGet_upgrades_downgrades and its wrapper function. The function retrieves upgrade and downgrade information for a given stock ticker from a specific API service.

Function Overview

The AlpGet_upgrades_downgrades function takes the following parameters:

  • ticker: A string representing the stock ticker you want to get information for.
  • api_token: Your API token as a string, which is required to access the service.

The function returns the ticker symbol if the operation is successful. In Excel, it writes the upgrade/downgrade data starting from the active cell.

Usage Example

Here is an example of how you might use this function in VBA with Excel:

            
Dim result As String
result = AlpGet_upgrades_downgrades("AAPL", "your_api_token")

In this example, replace "AAPL" with your desired stock ticker and "your_api_token" with your actual API token.

Expected Behavior

Once executed, the function makes a HTTP GET request to the specified server for the upgrade/downgrade data. The response is expected to be in JSON format with a data key. The function then writes:

  • Column headers based on the index provided in the response.
  • Corresponding data in subsequent rows.

Error Handling

If an error occurs during the process, a message box will display the error details, allowing users to troubleshoot the issue effectively.
