AlpGet Institutional Holders User Manual

Overview

The AlpGet_Institutional_Holders function retrieves information about the institutional holders for a given stock ticker from a local server and displays it in an Excel worksheet.

Function Signature

AlpGet_Institutional_Holders(ticker As String, api_token As String) As String

Parameters

  • ticker: The stock ticker symbol for which you want to fetch institutional holders data.
  • api_token: The API token required for authentication to access the data.

Return Value

Returns a string in the format ticker-api_token.

Usage Example

To use the AlpGet_Institutional_Holders function, insert the following code into your VBA editor:

' Set the active Excel cell where data output begins
Sub ExampleUsage()
Dim result As String
result = AlpGet_Institutional_Holders("AAPL", "your_api_token")
MsgBox "Function executed with result: " & result
End Sub

Output

The function will write the following information starting from the currently active cell:

  • Date Reported
  • Holder
  • Percentage Held
  • Shares
  • Value

Error Handling

If an error occurs during the execution, a message box will display the error description and line number.
