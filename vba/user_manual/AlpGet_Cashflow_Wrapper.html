User Manual for AlpGet_Cashflow

This document provides a guide on how to use the AlpGet_Cashflow function in VBA, which retrieves and displays cash flow data for a given stock ticker from a specified API endpoint.

Function: AlpGet_Cashflow

Description: This function acts as a wrapper for AlpGet_Cashflow_Wrapper. It fetches JSON-formatted cash flow data for a provided stock ticker by sending a GET request to a local API server, and writes the data into an active Excel worksheet starting from the active cell's row and column position.

Parameters

  • ticker (String): The stock ticker symbol for which cash flow data is requested.
  • api_token (String): A valid API token required for authentication with the API server.

Return Value

The function returns the input ticker symbol if successful, otherwise it may terminate if an error is encountered.

Example Usage

Dim ticker As String
Dim api_token As String
Dim result As String

ticker = "AAPL"
api_token = "your_api_token_here"

result = AlpGet_Cashflow(ticker, api_token)
MsgBox "Data fetched for ticker: " & result

Error Handling

The function includes basic error handling, which captures and displays an error message box if an operation within the function fails.

Dependencies

  • The function requires a reference to the Microsoft Scripting Runtime for JSON parsing capabilities.
  • Ensure that the JSON parsing library or module (e.g., JsonConverter) is included in your VBA project.

Notes

Make sure that the local API service is running and accessible at http://localhost:5000. Adjust the endpoint URL if different.
