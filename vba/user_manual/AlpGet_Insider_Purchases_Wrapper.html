User Manual for AlpGet_Insider_Purchases Function

Overview

The AlpGet_Insider_Purchases function retrieves insider purchase data for a specified stock ticker from a local server and populates it into an Excel spreadsheet starting from the active cell. It makes use of a helper function AlpGet_Insider_Purchases_Wrapper to perform the details of the data retrieval and Excel population.

Functionality

This function accesses a local API endpoint, sends a request formatted with a stock ticker and API token, and retrieves insider purchase data for the last 6 months. The retrieved data includes:

  • Shares
  • Sales
  • Net Shares Purchased (Sold)
  • Total Insider Shares Held
  • % Net Shares Purchased (Sold)
  • % Buy Shares
  • % Sell Shares

Usage Examples

Here is an example of how you can use the AlpGet_Insider_Purchases function in VBA:

    Sub GetInsiderPurchasesData()
        Dim ticker As String
        Dim api_token As String
        Dim result As String
        
        ticker = "AAPL"
        api_token = "your_api_token_here"
        
        result = AlpGet_Insider_Purchases(ticker, api_token)
        MsgBox result
    End Sub

This example sets the ticker to "AAPL" and sends a request to retrieve and display the insider purchase data for Apple Inc. You must replace your_api_token_here with a valid API token.

Error Handling

If an error occurs during the function execution, a message box will display the error details including the description and line number.
