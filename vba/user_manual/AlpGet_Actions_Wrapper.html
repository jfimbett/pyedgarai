AlpGet_Actions Function User Manual

Overview

The AlpGet_Actions and its helper function AlpGet_Actions_Wrapper are designed to fetch financial actions related to a specific stock ticker using a given API token. The data retrieved includes dividends and stock splits, and the result is populated into the active Excel sheet starting from the current active cell.

Function Signature

Function AlpGet_Actions(ticker As String, api_token As String) As String

Parameters

  • ticker: The stock ticker symbol as a string.
  • api_token: The API token as a string used for authentication to access the data.

Return Value

The function returns a string that combines the ticker and API token.

Usage Example

Sub ExampleUsage()
Dim result As String
' Call the function with your stock ticker and API token
result = AlpGet_Actions("AAPL", "your_api_token_here")
' Display the result
MsgBox result
End Sub

Notes

  • Ensure that Excel's active cell is where you want the data to be inserted, as data will be written starting from this position.
  • The function communicates with a local server running at http://localhost:5000. Ensure your environment has this server running with the necessary endpoints.
  • Please handle sensitive information like API tokens securely and ensure you have permission to access the data for the specified ticker.

Error Handling

If any error occurs during function execution, a message box will display the error description and the line number.
