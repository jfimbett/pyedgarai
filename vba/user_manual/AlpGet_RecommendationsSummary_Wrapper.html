AlpGet_RecommendationsSummary Function - User Manual

Function Overview

The AlpGet_RecommendationsSummary function retrieves a summary of recommendations for a specified stock based on its ticker. This function communicates with an API endpoint to fetch and display the summary of different recommendations (strong buy, buy, hold, sell, strong sell) for the specified ticker in the currently active Excel sheet starting from the selected cell.

Function Declaration

Function AlpGet_RecommendationsSummary(ticker As String, api_token As String) As String

Parameters

  • ticker: String - The stock ticker symbol you want to get the recommendations summary for.
  • api_token: String - Your API token for authentication with the recommendations API.

Return Value

The function returns the ticker symbol if the operation is successful, otherwise it returns an empty string if an error occurs.

Examples of Usage

' Example 1: Get recommendations summary for Apple Inc.
Dim ticker As String
ticker = "AAPL"
Dim apiToken As String
apiToken = "your_api_token"

Dim result As String
result = AlpGet_RecommendationsSummary(ticker, apiToken)
If result <> "" Then
    MsgBox "Recommendations summary retrieved for: " & result
Else
    MsgBox "Failed to retrieve recommendations summary."
End If

' Example 2: Get recommendations summary for Microsoft Corp.
result = AlpGet_RecommendationsSummary("MSFT", "another_api_token")
If result <> "" Then
    MsgBox "Recommendations summary retrieved for: " & result
Else
    MsgBox "Failed to retrieve recommendations summary."
End If

Error Handling

The function will display a message box with an error description in case of any issues during the API request or data processing:

An error occurred: [Error Description] [Line Number]

Notes

  • Ensure your API service is running locally and accessible at http://localhost:5000.
  • This function depends on the VBA JSON parser library (e.g., VBA-JSON) for parsing the JSON response.
  • Check your API permissions and rates to prevent throttling or access issues.
