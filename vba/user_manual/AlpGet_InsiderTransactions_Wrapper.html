AlpGet_InsiderTransactions Function User Manual

Description:

The AlpGet_InsiderTransactions function retrieves insider transaction data for a given stock ticker from a web service and populates the current Excel sheet starting from the active cell. It is a VBA function designed to interface with a local HTTP API and parse JSON responses.

Functions:

  • AlpGet_InsiderTransactions_Wrapper: A lower-level function that performs the actual API call and data handling.
  • AlpGet_InsiderTransactions: A wrapper function that invokes the AlpGet_InsiderTransactions_Wrapper function via the Evaluate method.

Parameters:

  • ticker (String): The stock ticker symbol for which insider transactions are requested.
  • api_token (String): The API token required for authenticating the request against the web service.

Usage:

Sub ExampleUsage()
    Dim ticker As String
    Dim api_token As String
    Dim result As String
    
    ticker = "AAPL"
    api_token = "your_api_token_here"
    
    result = AlpGet_InsiderTransactions(ticker, api_token)
    
    MsgBox "Transaction data has been added to the sheet for " & ticker
End Sub

Output:

The data filled into the Excel sheet will include the following columns with header titles:

  • Shares
  • URL
  • Text
  • Insider
  • Position
  • Transaction
  • Start Date
  • Ownership
  • Value

Note:

The function also includes error handling. If an error occurs during the API request or JSON parsing, a message box will display the error description and the line number where the error occurred.
