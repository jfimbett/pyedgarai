AlpGet_CompanyFacts User Manual

Description

The AlpGet_CompanyFacts function retrieves company financial data based on the given Central Index Key (CIK) using a specified API token. It makes an HTTP request to a local server and receives company facts in JSON format. The primary focus is on extracting share-related data and writing it to an active Excel worksheet.

Functions

AlpGet_CompanyFacts_Wrapper

This function communicates with an API to retrieve company facts and extracts data regarding outstanding common stock shares, which is then written to an Excel worksheet.

AlpGet_CompanyFacts

This function acts as a wrapper for AlpGet_CompanyFacts_Wrapper, allowing for easier invocation through Excel.

Parameters

  • cik: The Central Index Key representing the company for which you want to retrieve the data. It should be a string.
  • api_token: The API token required to authenticate the request. It should be a string.

Returns

A string in the format "cik-cik_value", indicating the CIK of the company for which data retrieval was attempted.

Examples of Usage

Excel VBA Code Example

' Example usage in Excel VBA
Dim result As String
result = AlpGet_CompanyFacts("0000320193", "your_api_token_here")
MsgBox result

Error Handling

The function is equipped with an error handler that provides a message box if any error occurs during execution, displaying the error description and the line number.
